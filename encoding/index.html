Signal Input

What Is Line Encoding?

Line encoding converts binary data into electrical, optical, or radio signals that can be transmitted across a communication medium.

What Is NRZ?

NRZ represents binary values using fixed signal levels. A logical one and logical zero correspond to different voltage levels without returning to a neutral state between bits.

What Is NRZI?

NRZI represents data through signal transitions rather than absolute voltage levels. A transition typically indicates a logical one while no transition indicates a logical zero.

What Is Manchester Encoding?

Manchester encoding guarantees a transition in the middle of every bit period. This provides excellent synchronization and clock recovery.

What Is 4B/5B Encoding?

4B/5B converts every four bits of data into a five-bit codeword chosen to ensure sufficient signal transitions.
